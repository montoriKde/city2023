開発目的：大学の授業

開発期間：2023年9月～12月

開発環境：
    ・OS：　Windows 11 Version 23H2
    ・エディタ：　Visual Studio Code
    ・使用言語：　Python C++

概要：
    テーマ「立体視用バーチャルシティ」
    四人チームで電子地図データを適切に処理し、後楽園周辺の都市空間を仮想世界（VRML）で表現する

    入力データ（与えられたデータ）：
        ・x,y,z座標が羅列されたファイル（約800万個）
        ・建物、地面の区別がない
        ・計測できていない点も存在する（特定のマイナス値で記録されている）

    要件：
        ・欠損点の補完
        ・地面と建物の判定
        ・データ削減



学生時代に学業面で力を入れたこと
チームで協力し、与えられた要件をもとに課題を半年間で解決する授業

（私の大学では画像・映像コンテンツ演習という経済産業省から「社会人基礎力を育成する授業 30 選」に選ばれている授業があります。この授業は人数に上限があり、意欲と成績が必要ですが、私は学業に力を入れており、GPAが3.5以上あるため参加することができました。）

座標と高さの点データと要件をもとにVRML化して地形を再現するという課題に取り組みました。私はスケジュール管理と要件を満たすための設計に注力しました。スケジュール管理では、全工程を細かくタスクに分け、各メンバーの役割を明確にしました。定期的に進捗を確認し、遅延が発生した際には迅速に話し合いを行い、解決策を見出すことでプロジェクトのスムーズな進行を確保しました。また、プログラムの概要を決める設計では、積極的に案を出しつつ、図を用いて案をまとめることでメンバーがプログラムを理解し、コードを書きやすくなるように工夫しました。この経験から基礎を築く重要さと、面白さを体感しました。
